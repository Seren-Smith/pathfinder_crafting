# Pathfinder Weapon & Armor Crafting Calculator

A web-based calculator for determining costs and requirements for crafting magical weapons and armor in Pathfinder RPG.

## Features

- Calculate crafting costs for both weapons and armor
- Select from all standard base items
- Choose from complete lists of weapon and armor enchantments
- Support for special materials (Adamantine, Mithral, Dragonhide, etc.)
- Detailed breakdown of:
  - Material costs
  - Enhancement costs
  - Masterwork costs
  - Crafting DCs
  - Required caster level
  - Crafting time (normal and accelerated)

## How to Use

1. Select whether you're crafting a weapon or armor
2. Choose your base item from the dropdown
3. Set your existing and desired enhancement bonuses
4. Select any existing enchantments the item already has
5. Select the new enchantments you want to add
6. Choose your material type
7. Check the appropriate boxes if you know all required spells or if the item is already masterwork
8. Click "Calculate" to see the full crafting details
